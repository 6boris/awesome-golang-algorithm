# Section

The legacy `{{</* section */>}}` shortcode (which renders pages in a section as
a definition list) is deprecated and will be removed in a future hugo-book
release. Build a manual list of child pages with regular markdown instead:

```md
- [First page](first-page)
  First page summary...

- [Second page](second-page)
  Second page summary...
```

## Example

- [First page](first-page)

  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
  tempor incididunt ut labore et dolore magna aliqua.

- [Second page](second-page)

  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
  tempor incididunt ut labore et dolore magna aliqua.
