# 0082.Remove-Duplicates-from-Sorted-List-II

## Description

Given a sorted linked list, delete all nodes that have duplicate numbers, leaving only distinct numbers from the original list.

**Example 1:**

```text
Input: 1->2->3->3->4->4->5
Output: 1->2->5
```

**Example 2:**

```text
Input: 1->1->1->2->3
Output: 2->3
```

**Tags:** Linked List

## 题意

> 给定一个有序的单链表，删除其中的重复元素。
