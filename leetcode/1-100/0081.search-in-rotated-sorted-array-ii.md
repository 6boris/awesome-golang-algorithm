# 0081.Search-in-Rotated-Sorted-Array-II

## Description

Given two binary strings, return their sum \(also a binary string\).

The input strings are both **non-empty** and contains only characters `1` or `0`.

**Example 1:**

```text
Input: a = "11", b = "1"
Output: "100"
```

**Example 2:**

```text
Input: a = "1010", b = "1011"
Output: "10101"
```

**Tags:** Math, String

## 题意

> 给你两个二进制串，求其和的二进制串。
